## 📌 Project Overview
This repository contains the complete workflow for modeling and forecasting COVID-19 transmission dynamics across Indian states. By integrating localized epidemiological case counts (JHU CSSE) with human movement shifts (Google Community Mobility Data), this project leverages an **Enhanced Graph Neural Stochastic Differential Equation (GN-SDE)** to capture both the deterministic spread and stochastic volatility of the virus.

## 🗂️ Repository Contents
* **`Model_Workflow.ipynb`**: The main Jupyter Notebook containing the data preprocessing pipeline, empirical spatial graph construction, PyTorch GN-SDE architecture, training loop, and visual evaluation.
* **`ARTPARK_Report.pdf`**: The formal LaTeX write-up detailing the mathematical foundations, architectural choices, and final performance metrics of the model.
* **`India_COVID_Cleaned.csv`**: Preprocessed state-level COVID-19 dataset featuring 7-day rolling averages and geographic coordinates.
* **`India_Mobility_Cleaned.csv`**: Preprocessed state-level Google mobility dataset, specifically isolating shifts in workplace movement from baseline.
* **`ARTPARK_Model_Results.png`**: Visual diagnostics generated by the model, including the training loss curve and the out-of-sample 7-day forecast comparison.

## 🧠 Methodology
Traditional discrete-time RNNs and deterministic SIR models struggle with the inherent randomness of human behavior during a pandemic. This project uses a continuous-time SDE approach:
1. **Spatial Graph**: An empirical adjacency matrix is built using an RBF kernel on state coordinates, defining structural transmission corridors.
2. **Drift Network**: A multi-layer Graph Convolutional Network (GCN) models the deterministic expected spread.
3. **Diffusion Network**: A secondary GCN scales Brownian motion to simulate stochastic uncertainties (e.g., superspreader events or sudden containment lapses).
